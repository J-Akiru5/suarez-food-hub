Monorepo overview, tech stack, and deployment notes (Vercel)

Quick summary

This monorepo contains multiple Next.js apps (web, admin, rider, staff), shared packages, Prisma for Postgres, and is deployed to Vercel.

Deployment & Run Notes

  • Deploy target: Vercel — apps contain `vercel.json` and Next configs.
  • Local dev: run `pnpm dev` from repo root to start Turbo-powered dev for all apps.
  • Database: Prisma is configured at `prisma/schema.prisma`. Use `pnpm db:migrate` / `pnpm db:seed`.
