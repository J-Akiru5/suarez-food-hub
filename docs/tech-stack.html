Deployment & Run Notes

  • Deploy target: Vercel — apps include `vercel.json` and Next configs.
  • Local dev: run pnpm dev at repo root to start Turbo-powered dev.
  • Database: Prisma is configured at prisma/schema.prisma. Use pnpm db:migrate / pnpm db:seed.
